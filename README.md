# Internship-Task-1
Task 01: Company Website Frontend Development
________________________________________
1. Task Title
Company Website Frontend Development using WordPress
________________________________________
2. Objective
The objective of this task was to design and develop the frontend of the company website using WordPress CMS hosted on Hostinger. The goal was to create a responsive, clean, and professional website interface with structured navigation, dropdown menus, contact form integration, and SEO-friendly structure.
________________________________________
3. Platform & Tools Used
•	WordPress CMS
•	Hostinger Hosting Platform
•	Astra Theme
•	WPForms Plugin
•	Rank Math SEO Plugin
•	LiteSpeed Cache Plugin
________________________________________
4. Task Requirements
The website was required to fulfill the following functional requirements:
•	Responsive design (mobile and desktop compatible)
•	Clean and professional user interface
•	Easy navigation with dropdown and sub-dropdown menus
•	Contact form integration
•	SEO-friendly page structure
•	Fast loading performance

6. Implementation Process
Step 1: WordPress Setup
•	Logged into Hostinger and accessed WordPress dashboard.
•	Installed and activated Astra theme for lightweight and professional design.
Step 2: Page Creation
•	Created all main pages including Home, About Us, Contact Us, and Testimonials.
•	Developed category pages and corresponding Products and Services subpages.
Step 3: Navigation Menu Design
•	Created structured navigation menu from Appearance → Menus.
•	Implemented dropdown and sub-dropdown hierarchy using drag-and-drop functionality.
Step 4: Contact Form Integration
•	Installed WPForms plugin.
•	Created a simple contact form.
•	Embedded the form into the Contact Us page.
Step 5: SEO Configuration
•	Installed Rank Math SEO plugin.
•	Configured basic SEO settings for improved search engine visibility.
Step 6: Performance Optimization
•	Installed LiteSpeed Cache plugin.
•	Optimized website loading speed.
